# Backend Learning Repository – Node.js, Express & MongoDB

This repository contains the concepts, practice code, and exercises I completed during a 3-week backend development course.  

⚠️ This is not a full production project. It is a structured collection of backend fundamentals and hands-on practice implementations as part of my learning journey.

---

## What I Learned

During this course, I focused on building a strong foundation in backend development using modern JavaScript technologies.

### Core Technologies
- Node.js (Runtime Environment)
- Express.js (Web Framework)
- MongoDB (Database)
- Mongoose (ODM for MongoDB)

---

## Topics Covered

- Basic server setup using Node.js
- Creating routes with Express
- Understanding HTTP methods (GET, POST, PUT, DELETE)
- Building RESTful API structure
- Implementing CRUD operations
- Connecting server to MongoDB
- Schema modeling with Mongoose
- Middleware usage
- Error handling
- Request validation
- Managing environment variables
- Handling asynchronous operations
